Як працювати

  1. Ввести свій код маркетолога в поле marketer ID
  2. Скопіювати значення для Domen, Type, Product info з mastersheet (можна скопіювати всі за один раз)
  3. Заповнити інші поля вручну
  4. Натиснути "Generate Links"
  5. Натиснути "Copy link" / "Copy Unsub Link" в рядку з потрібним посиланням або "Copy All Links"
  6. Вставити посилання в адресну строку і перевірити, що немає помилок

Поля не скидаються при оновленні або закритті вкладки, якщо вам потрібно скинути значення, натисніть "Clear All Fields", що очистіть ВСІ поля в таблиці.
